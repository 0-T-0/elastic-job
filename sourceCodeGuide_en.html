Source codes compile problems

Because of focus extremely simple code, elastic-job use Lombok. If you read source codes meet the problem for @Getter, @Setter annotation cannot compile, please install Lombok to your IDE following next steps. Use release jar only please ignore.

Lombok install guild

  • Eclipse

    Double click lombok.jar file.

    Confirm installation directory for Eclipse, click install/update button and restart Eclipse. Finish!

    How to confirm installation success?
    Confirm Eclipse directory has lombok.jar file, and file eclipse.ini has content:
    -javaagent:lombok.jar  -Xbootclasspath/a:lombok.jar
