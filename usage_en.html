Elastic-job

Elastic-Job is a distributed scheduled job framework, based on Quartz and Curator.

View on GitHub Download .zip Download .tar.gz

Usage

  • Install Java environment

    Please use JDK1.7 or above.

    Reference http://www.oracle.com/technetwork/java/javase/downloads/index.html

  • Install Zookeeper

    Please use Zookeeper 3.4.6 or above.

    Reference https://zookeeper.apache.org/doc/trunk/zookeeperStarted.html

  • Install Maven

    Please use Maven 3.0.4 or above.

    Reference http://maven.apache.org/install.html

  • Add dependency for elastic-job

<!-- add dependency for elastic-job core module -->
<dependency>
    <groupId>com.dangdang</groupId>
    <artifactId>elastic-job-core</artifactId>
    <version>1.0.2</version>
</dependency>

<!-- add dependency for elastic-job spring module, can use customised namespace -->
<dependency>
    <groupId>com.dangdang</groupId>
    <artifactId>elastic-job-spring</artifactId>
    <version>1.0.2</version>
</dependency>
